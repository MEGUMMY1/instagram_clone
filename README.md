# 4팀 FE조혜진 | BE이호준, 윤준섭

- 인스타그램 PC버전을 기준으로 함
- 필수 기능
    - 회원 기능 JWT
        - 로그인 / 로그아웃
    - 게시물 CRUD(Create, Read, Update, Delete)
        - 사진은 우선 이미지 Url 을 받는 형식으로 제작
        - 추후 AWS S3를 이용해 리펙토링
    - 게시물 댓글
    - 게사물 좋아요
    - 게시물 검색
    - 스토리, DM등 부가 기능은 구현 필요 X
